# HGAM_poptrends
Use HGAMs and similar models to estimate population trends from subpopulation count data

Code provided for the manuscript "How to estimate overall population trends when sites show varying
population counts over time?"

* code_for_simulationstudy.R: code for the simulation study conducted in the manuscript (all models except TRIM, see sim_trim.T)
* mmyo_example.txt: toy dataset showing the structure of the real-world dataset used in the manuscript, this is not real data!
* code_for_mmyo.R: code for the case study conducted in the manuscript
* sim_trim.R: code the simulation study conducted in the manuscript using TRIM
